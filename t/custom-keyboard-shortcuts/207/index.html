I’ve sent this to our product team as a feature request!

We do have some keyboard shortcuts that I wanted to share:

  • CTRL + C - Can be used to copy controls, actions, and even web pages to your clipboard
  • CTRL + V - Can be used to paste those controls, actions, etc… in studio
  • CTRL + S - Save your app
  • CTRL + K - Universal search in studio
