Hey Chris! what does accountlist look like?

You can order objects in an array by using something similar to what you have and using ASCENDING or DESCENDING

So if i have airData.results that looks like this:

[
    {
      "__id": "",
      "Name": "test"
    },
    {
      "__id": "",
      "Name": "hey"
    },
    {
      "__id": "",
      "Name": "life"
    }
  ]

Then i would do something like this to get them to alphabetized.

FROM
  item
IN
  airData.results
ORDER BY
  item.Name
ASCENDING
SELECT
  item
