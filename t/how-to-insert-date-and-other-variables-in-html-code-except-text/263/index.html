So dates, time, and datetime data types in Airkit are actually objects, which means that html body won’t be able to parse it.

What you need to do is use an airscript function to change that datetime into a text.

So you can use something like:


FORMAT_DATETIME(  
dateinputvariable,   
  "MMMM Do, YYYY h:mm A z"
)

For more information, you can check out the two articles below:

https://support.airkit.com/reference/format_datetime
