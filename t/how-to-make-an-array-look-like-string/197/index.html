How to make an array look like string?

kateocampo

How can I format my array like below?

Array: [“Engineered wood”,“Tile”]

Output should be: Engineered wood, Tile

aldren

This sounds like the perfect use for the JOIN function!

It takes both an array (list) and a separator as input, and returns the resulting string. For instance:

JOIN([“Engineered wood”,“Tile”], ", ") returns "Engineered wood, Tile"
