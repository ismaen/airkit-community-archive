darren55

I know this is a general question but wanted to start a thread on this as I find myself tinkering at the styles to make my app look nice. Anyone have any good tips and tricks for styling?

I’ll share one myself to get things started:

When I am using a container list and want to evenly distribute the different items in my container list, I set the parent container to 100% width. Then controls nested in the container, I set the width of each control to add up to 100%.

For example, I want the rows in my container list to be even across my container list:

So I set the parent container to distribute horizontal, and 100% width:
