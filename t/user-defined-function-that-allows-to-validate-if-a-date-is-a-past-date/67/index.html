To check to see if a date is a past date, you can compare two date objects using the comparison operators <,>,<=,>=,=. What you can do is compare the current date using DATE_FROM_DATETIME(NOW()) to the date object you are wanting to compare against.

Below is how I structured this User Defined Function so that it takes two parameters. A date parameter and a timezone parameter.

IF(
  date < DATE_FROM_DATETIME(NOW(time_zone)),
  TRUE,
  FALSE
)
