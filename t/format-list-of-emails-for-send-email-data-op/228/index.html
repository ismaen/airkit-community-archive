Format list of emails for Send Email data op

Mike_Cammon

What’s the best way to format a list in the email data op so it appears listed vertically, instead of as one line?

I used a transform to flatten the list, but I’m not sure how to modify it for the email.

Example:
(what I want)
email1
email2
email3

(what I get) email1 email2 email3

ismaen

Hi @Mike_Cammon! Great questions! Are your emails currently in an array? Do they look something like this?

["ismaen@airkit.com","ismaen2@airkit.com","ismaen3@airkit.com"]

If yes, then you can use the Send Email data operation, and in the Body HTML property, you would pass this expression.

"<p> {{JOIN(array_of_emails, "<br>")}} </p>"

The output will look like this:

<p> ismaen@airkit.com <br> ismaen2@airkit.com <br> ismaen3@airkit.com </p>

which puts a line break after each email.
