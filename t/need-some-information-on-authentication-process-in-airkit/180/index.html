Hi @tanmay.sardar! What are you trying to authenticate?

You can authenticate a journey using a few different methods:

  • OAuth 2.0 - Auth0
  • OAuth 2.0 - Google
  • OAuth 2.0 - Okta

and what you might be looking for Authenticating using Azure AD B2C.

The way that this works is that you can use Azure AD B2C as the OAuth vendor and then leverage any SAML provider that you’d like to authenticate, with Microsoft being one of them.
