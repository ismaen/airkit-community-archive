Redirecting to external 404 webpage link

chandu.singh

I am in a situation where I have to redirect to an external link to show 404 msg based on some condition
otherwise, I need to redirect to my AIRKIT app pages
Is it possible through AIRKIT?

dinesh

Hi Chandu,

You can use the Airkit API trigger feature to achieve this.

In the API definition, you can use a transform step to specify the condition and set the corresponding URL where you’d like to navigate the user (this can also be an Airkit Journey URL).

The response from the API should be a 302 response with an appropriate Location URL as in the attached image.
