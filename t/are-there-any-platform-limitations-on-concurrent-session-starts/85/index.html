Are there any platform limitations on concurrent session starts?

Drew

Looking to start ~750 sessions at once that send a canvas link and I am wondering if there are any limitations to be aware of.

joechrz

While there’s no strict limit you may see a very small percentage of those sessions fail to start. We’d recommend starting your sessions in smaller batches (~250) to be safe.

We’re in the process of building out a feature specifically for this use case, so pretty soon you’ll be able to start thousands of sessions without any issues.

joechrz

Since last year, Airkit has also released the ability to “queue up” your journey starts. Using this feature you can start thousands of sessions without worrying about load. See Start Journey
