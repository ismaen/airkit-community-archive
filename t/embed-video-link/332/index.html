Hi Mike,

This is possible using the Embedded External Content Control.

However, most of the standard video providers I tried (YouTube, Vimeo, Twitter) all didn’t work because they want to read/write cookies on the document, which we don’t currently allow for security reasons. (I think they use these cookies to make sure users aren’t abusing the Embed feature or putting it on sites they don’t approve).
