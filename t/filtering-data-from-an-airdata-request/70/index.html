Hi @dinesh,

When querying data in AirData using the QUERY operation, you have to use specific syntax to create your query. This can be found in the doc AirData Querying Capabilities.

For example, if you wanted to find all the records that match a specific email address (i.e. ismaen@airkit.com) you would have to do the following:

Example Data:

Query:

{ "email": { "eq": "ismaen@airkit.com" } }

2022-01-18_14-59-03900×441 15 KB

This would result in only records where the email column is equal to “ismaen@airkit.com”
