App Setting Variable for Hyperlink control Href

Mike_Cammon

is it possible to have a hyperlink (displayed as a button on a form) pull the URL from a global variable thats been converted to an app setting? so in this case i have a variable session.huburl, and each environment would have a different URL that a hyperlink should navigate too.

i go to set Href in the hyperlink control as profile.huburl, but it doesnt accept the value. how can i get the app setting variable set for that hyperlink?

joechrz

Hey Mike

This is possible- rather than typing directly into the input, you have to drop into “expression mode” to use an Airscript variable. Typing directly into the text field will always treat your input as literal text, so just click on the chevron and use your variable there. Remember to remove any quotes!
