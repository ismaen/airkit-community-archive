Preparing CSS for HTML to PDF

nico

Currently Airkit uses PhantomJS headless browser to render the HTML content to create a PDF file via HTML to PDF data operation. To make CSS compatible with PhantomJS you can run your CSS via AutoPrefixer targeting “PhantomJS 1.9” in the browserlist field. This will make CSS compatible with PhantomJS rendering engine!
