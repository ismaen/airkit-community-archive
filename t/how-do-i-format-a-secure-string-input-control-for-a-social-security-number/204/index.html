Hey @grakozy! At the moment, this doesn’t seem possible as there isn’t a formatting option for SSN using the secure string input. An additional validation you can use though, is the maximum length property and pass “7” to it so the user provides you a valid SSN, as well as making sure the input mode is numeric. You can also do additional validation so that the value is only 7.
